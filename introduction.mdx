## Overview

Peter Parker provides a centralized hub for your project documentation. You create organized spaces, manage content collaboratively, and publish polished docs with ease. This platform streamlines workflows, ensuring your team stays aligned with up-to-date information.

<Callout kind="info">
Peter Parker supports MDX for rich, interactive documentation pages with components like Cards and Steps.
</Callout>

## Key Features

Peter Parker offers powerful tools to enhance your documentation experience.

<Columns cols={3}>

## Quick Start

Get up and running in minutes with these steps.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at `https://peterparker.dev` using your email or GitHub account.
  </Step>
  <Step title="New Space" icon="plus">
    Click **New Space** and name it for your project, like "MyApp Docs".
  </Step>
  <Step title="Add Content" icon="edit-3">
    Create your first page with the MDX editor. Use components for interactive content.
  </Step>
  <Step title="Publish" icon="upload-cloud">
    Preview and publish. Share the live URL with your team.
  </Step>
</Steps>

## Embed Peter Parker Docs

Integrate docs into your app with a simple script.

<CodeGroup tabs="JavaScript,React">
  ```javascript
  // Add to your HTML head
  const script = document.createElement('script');
  script.src = 'https://peterparker.dev/embed.js?space=myapp-docs';
  script.async = true;
  document.head.appendChild(script);
  ```
  ```jsx
  // React component
  import { PeterParkerEmbed } from '@peterparker/embed';

  function DocsEmbed() {
    return (
      <PeterParkerEmbed spaceId="myapp-docs" height="600px" />
    );
  }
  ```
</CodeGroup>
