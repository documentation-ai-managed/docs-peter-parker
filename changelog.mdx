## Recent Updates

Stay informed about new features, bug fixes, and improvements. Peter Parker releases updates regularly to enhance your documentation experience.

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "bugfix"]}>

## New Features

- Added real-time collaboration for teams, allowing multiple users to edit docs simultaneously.
- Introduced custom themes with your brand color `#3B82F6` support for personalized spaces.
- New search filters let you find docs by tags, authors, or update date.

## Bug Fixes

- Fixed rendering issues with nested code blocks in MDX files.
- Resolved authentication errors during session timeouts.
- Improved mobile responsiveness for table views.

## Improvements

- Faster page loads by 40% with optimized asset caching.
- Better keyboard navigation in the editor.

</Update>

## How to Upgrade

You upgrade Peter Parker by pulling the latest version. Follow these steps to ensure a smooth update.

<Steps>
  <Step title="Check Current Version" icon="package">
    Run the version check command to see your installed version.

````bash
peterparker --version
````
  </Step>
  <Step title="Backup Your Data" icon="database">
    Export your docs before upgrading.

    <CodeGroup tabs="CLI,UI">
````bash
peterparker export --all > backup.json
````
````json
// Use dashboard export at https://dashboard.example.com/export
{
  "docs": [...]
}
````
    </CodeGroup>
  </Step>
  <Step title="Install Update" icon="download">
    Update via your package manager.

    <Tabs>
      <Tab title="npm" icon="package">
````bash
npm update peterparker
````
      </Tab>
      <Tab title="Docker" icon="docker">
````bash
docker pull peterparker:latest
docker run -p 3000:3000 peterparker:latest
````
      </Tab>
    </Tabs>
  </Step>
  <Step title="Verify" icon="check-circle">
    Restart and test key features like editing and search.
  </Step>
</Steps>
