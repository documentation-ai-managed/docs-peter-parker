## Welcome

Peter Parker lets you organize your project docs in one place.

## Quick Start

1. Sign in at `https://peterparker.com`
2. Click "New Project" and name it
3. Choose a documentation template
4. Start adding pages or import Markdown files

## Next steps

Explore project settings to invite team members and customize your space.
